# História em Jogo

Sistema educativo que combina narrativas históricas, cenas ilustradas e minigames.

## Conteúdo disponível

- História do Brasil
  - Capítulo 2: Travessia do Atlântico e Volta do Mar

## Abrir localmente

Abra o arquivo `index.html` no navegador.

## Publicar no GitHub Pages

1. Crie um repositório chamado `historia-em-jogo`.
2. Envie todo o conteúdo desta pasta para a raiz do repositório.
3. No GitHub, abra `Settings` → `Pages`.
4. Em `Build and deployment`, selecione `Deploy from a branch`.
5. Escolha a branch `main` e a pasta `/ (root)`.
6. Salve e aguarde a publicação.

O `index.html` principal abrirá o sistema, e os minigames permanecerão organizados dentro de `conteudos/`.
